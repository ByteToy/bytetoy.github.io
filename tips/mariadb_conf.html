Armbian安装Mysql(MariaDB)数据库

debian9默认情况下，Mysql数据库已经替换为MariaDB数据库，配置方式已经同原Mysql不同。 通过apt安装mysql-server后，需要进行一些列的配置，才能成功登录

一、安全配置

运行脚本mysql_secure_installation，无需密码，进行部分按权设置

  1. 配置root密码
  2. 删除匿名帐号
  3. 启用root的远程登录
  4. 删除test数据库
  5. 立即刷新数据库权限
bytetoy@aml:~$ sudo mysql_secure_installation

二、权限配置

按权配置后，可以使用mysql直接登录，然后可以新建帐号和配置权限

  1. 新建帐号:GRANT ALL PRIVILEGES on *.* TO 'admin'@'%' IDENTIFIED BY '9527' WITH GRANT OPTION;
  2. 设置帐号密码：SET PASSWORD FOR admin=PASSWORD('9527');
  3. 刷新权限：FLUSH PRIVILEGES;
bytetoy@aml:~$ sudo mysql

四、配置帐号远程登录

  1. 修改配置文件：
    sudo vim /etc/mysql/mariadb.conf.d/50-server.cnf
  2. 修改绑定的IP地址，然后重启mysql服务
    bind-address=0.0.0.0  #原为127.0.0.1
systemctl restart mysqld
